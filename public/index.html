Family Trouble

Settle the family dispute the old-fashioned way — in a house full of hallways and nowhere to hide. Share the room code to invite family members.

Leave empty to create a new room, or enter a 6-character code to join.

Game mode

1 — online from this machine (or open other tabs / invite with the room code). 2–4 — couch co-op: extra players on the same keyboard (see controls in-game).

Customize keyboard (per seat)

Click a control, then press a key. Hold Alt while pressing to add a second key for the same action (for example WASD and arrows together). Escape cancels.
